    - getBalance(address) returns { sol: number, usdc: number } from Solana RPC
    - Wallet path is fixed at ~/.mainlayer/wallet.json (per D-12)
    - encrypt then decrypt round-trips to identical bytes
  </behavior>
  <action>
    1. Create src/services/wallet-service.ts with class WalletService:

    **Constants:**
    - `WALLET_PATH = join(homedir(), '.mainlayer', 'wallet.json')`
    - `PBKDF2_ITERATIONS = 200_000`
    - `USDC_MINT = 'EPjFWdd5AufqSSqeM2qN1xzybapC8G4wEGGkZwyTDt1v'` (Solana mainnet USDC)

    **Private methods:**
    - `encryptKeystore(privateKeyBytes: Uint8Array, pubkey: string, passphrase: string): Promise<EncryptedKeystore>`
      - salt = randomBytes(16), iv = randomBytes(12)
      - key = await pbkdf2Async(passphrase, salt, 200_000, 32, 'sha256')
      - cipher = createCipheriv('aes-256-gcm', key, iv)
      - encrypted = Buffer.concat([cipher.update(privateKeyBytes), cipher.final()])
      - authTag = cipher.getAuthTag()
      - Return { version: 1, pubkey, salt: hex, iv: hex, ciphertext: hex, authTag: hex }

    - `decryptKeystore(keystore: EncryptedKeystore, passphrase: string): Promise<Uint8Array>`
      - Derive key same way, createDecipheriv, setAuthTag BEFORE update/final
      - If auth tag fails, catch and throw AppError('Invalid passphrase', EXIT_CODES.AUTH_ERROR)

    - `readKeystore(): Promise<EncryptedKeystore>`
      - Read WALLET_PATH, JSON.parse, validate version === 1
      - If file not found, throw AppError('No wallet found. Run: mainlayer wallet create', EXIT_CODES.NOT_FOUND)

    - `writeKeystore(keystore: EncryptedKeystore): Promise<void>`
      - Ensure ~/.mainlayer/ dir exists (mkdirSync recursive)
      - writeFileSync(WALLET_PATH, JSON.stringify(keystore, null, 2), { mode: 0o600 })

    **Public methods:**
    - `async create(passphrase: string): Promise<string>`
      - Check if WALLET_PATH exists; if so, throw AppError('Wallet already exists', EXIT_CODES.ALREADY_EXISTS)
      - Generate extractable Ed25519 keypair: `crypto.subtle.generateKey({ name: 'Ed25519' }, true, ['sign', 'verify'])`
      - Export private key: `new Uint8Array(await crypto.subtle.exportKey('pkcs8', keyPair.privateKey))`
      - Strip PKCS8 header (first 16 bytes) to get 32-byte raw key
      - Get address: `await getAddressFromPublicKey(keyPair.publicKey)` from @solana/kit
      - Encrypt and write keystore
      - Return address

    - `async importFromBase58(base58Key: string, passphrase: string): Promise<string>`
      - Decode base58 to bytes (use bs58 or manual)
      - Use `createKeyPairSignerFromBytes(bytes)` to validate and get address
      - Encrypt raw key bytes and write keystore
      - Return address

    - `async importFromMnemonic(mnemonic: string, passphrase: string): Promise<string>`
      - Use `@scure/bip39` to validate mnemonic and derive seed
      - Extract first 32 bytes of seed as Ed25519 private key
      - Use `createKeyPairSignerFromBytes(seedBytes)` to get address
      - Encrypt and write keystore
      - Return address

    - `async getAddress(): Promise<string>`
      - Read keystore, return keystore.pubkey — NO decryption (per WALL-07)

    - `async exportPrivateKey(passphrase: string): Promise<string>`
      - Read keystore, decrypt, encode as base58, return

    - `async getBalance(): Promise<{ sol: number; usdc: number }>`
      - Get address from keystore
      - Create Solana RPC: `createSolanaRpc(configService.get('solanaRpcUrl') ?? process.env.MAINLAYER_SOLANA_RPC_URL ?? 'https://api.mainnet-beta.solana.com')`
      - SOL: `rpc.getBalance(address(addr)).send()` → lamports / 1e9
      - USDC: `rpc.getTokenAccountsByOwner(address(addr), { mint: address(USDC_MINT) }, { encoding: 'jsonParsed' }).send()` → uiAmount or 0
      - Return { sol, usdc }

    Export: `export const walletService = new WalletService()`

    2. Write tests/services/wallet-service.test.ts:
       - Use tmp dir for wallet path (vi.mock or pass path as constructor arg)
       - Test encrypt/decrypt round-trip: random 32 bytes survive encryption + decryption
       - Test wrong passphrase throws AppError with AUTH_ERROR code
       - Test create writes file with 0o600 permissions
       - Test create when file exists throws ALREADY_EXISTS
       - Test getAddress reads pubkey without decrypting
       - Do NOT test getBalance (requires network) — mock RPC in integration tests later
