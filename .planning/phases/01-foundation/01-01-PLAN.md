       - Uses `conf` library with `cwd: join(homedir(), '.mainlayer')` and `configName: 'config'` (per D-09, Pitfall 8)
       - Schema type is MainlayerConfig from src/types/config.ts
       - Methods:

         - `set(key: ConfigKey, value: string): void` — config.set(key, value)
         - `delete(key: ConfigKey): void` — config.delete(key)
         - `getAll(): MainlayerConfig` — config.store
         - `getApiUrl(): string` — returns `this.get('apiUrl') ?? process.env.MAINLAYER_API_URL ?? 'https://api.mainlayer.io'` (per INFRA-05)
         - `clear(): void` — clears all config (used by auth logout)
       - Export singleton: `export const configService = new ConfigService()`

    2. Create src/cli/config.ts:
       - `export function configCommand(): Command`
       - Subcommand `get <key>`:
         - Validates key is in KNOWN_CONFIG_KEYS; if not, printError and exitCode=VALIDATION_ERROR
         - Prints value via formatOutput (plain text single value per D-04, JSON when --json)
       - Subcommand `set <key> <value>`:
         - Validates key is in KNOWN_CONFIG_KEYS
         - Calls configService.set(key, value)
         - Prints success message via printSuccess (per D-06)
       - Both accept --json flag from parent
